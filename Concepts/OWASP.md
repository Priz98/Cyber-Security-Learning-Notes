# OWASP

## What is OWASP?

OWASP is an organisation that mantains a list of documentations for majority of the security incidents that we have encounter in the past.
It provides a reference to learn th avoid the mistakes or vulnerabilties made in past to secure or product.
It also provides the best practices that a developer should do in order to make a secure product.
